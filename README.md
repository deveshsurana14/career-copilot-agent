# Career Copilot 🚀

AI-powered career advisor built for the Kaggle + Google 5-Day AI Agents Intensive.

## Features

* Gemini-powered career guidance
* Persistent session memory
* Career goal tracking
* Skills analysis
* Streamlit-based UI

## Tech Stack

* Python
* Streamlit
* Google Gemini API
* Session Memory

## Setup

### Clone Repository

git clone https://github.com/deveshsurana14/career-copilot-agent.git

cd career-copilot-agent

### Create Virtual Environment

python -m venv venv

### Activate Environment

Windows:

venv\Scripts\activate

### Install Dependencies

pip install -r requirements.txt

### Configure API Key

Create a `.env` file:

GEMINI_API_KEY=YOUR_API_KEY

### Run Application

streamlit run app.py
